# 🐛 Bug Tracker 

## 📌 Overview
Bug Tracker Pro is a Java Swing desktop application designed to simulate a real-world bug tracking system used in software development teams.  
It supports multiple roles including **Admin, Project Manager, Developer, and Tester**, each with specific permissions and responsibilities.

The system helps manage bugs throughout their lifecycle, assign tasks, and improve team collaboration with a modern interactive UI.

---

##  Features
- Role-Based Access Control (Admin, Project Manager, Developer, Tester)
-  Bug Management (Add, Update, Delete, Track)
- Bug Status Tracking (Open, In Progress, Closed)
- Bug Assignment to Developers
- User Management System (CRUD operations)
- Modern UI with animations and custom styling
- File-based data persistence
- Interactive dashboards for each role

---

## Roles & Responsibilities

### Admin
- Manage all users (Add / Update / Delete)
- View all bugs in the system

### Project Manager
- Assign bugs to developers
- Monitor project progress

###  Developer
- View assigned bugs
- Update bug status (In Progress / Fixed)

###  Tester
- Report new bugs
- Verify fixes and close bugs

---

##  System Design
The system follows Object-Oriented Programming principles:

- Separation between GUI and Business Logic
- `Admin` class handles core operations (users & bugs)
- Data stored using file handling (`users.txt`, `bug.txt`)
- Modular design with reusable components

---

## 🧪 Example Workflow
1. Tester reports a new bug  
2. Project Manager assigns it to a Developer  
3. Developer fixes the bug and updates status  
4. Tester verifies and closes the bug  

---

## 🛠️ Tech Stack
- Java (Core Java)
- Java Swing (GUI)
- Object-Oriented Programming (OOP)
- File Handling (TXT storage)
